## Barbershop fullstack app
### Implemented with:
<ul>
<li><b>Node.js<b></li>
<li><b>Express<b></li>
<li><b>MongoDB<b></li>
<li><b>React<b></li>
</ul>

## To run project use: 
To install <i>server</i> dependencies
### `npm install `
To install <i>client</i> dependencies
### `npm run client install `
To run both
### `npm run dev `
Server is automatically updating with <i>nodemon</i>
